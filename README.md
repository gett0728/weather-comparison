# 日本気象比較サイト

地方への移住を検討する際に、複数の候補地の気象データを比較できるWebアプリケーションです。
最大3地点を同時に選択し、過去20年分の気象データをグラフとテーブルで視覚的に比較できます。


## 特徴

- **複数地点の同時比較**: 最大3地点の気象データを並べて表示
- **豊富な気象指標**: 気温・降水量・日照時間・湿度・風速・天候日数を確認可能
- **長期データ対応**: 過去20年分のデータで長期的な傾向を把握
- **直感的なUI**: グラフとテーブルで分かりやすく表示

## 確認できる気象データ

| 項目 | 詳細 | 備考 |
|------|------|------|
| 気温 | 平均気温、最高気温、最低気温 | - |
| 降水量 | 月間・年間降水量 | - |
| 日照時間 | 月間・年間日照時間 | - |
| 湿度 | 平均湿度 | 3ヶ月以前のデータは取得不可 |
| 風速 | 平均風速 | - |
| 天候日数 | 晴天日数、雨天日数など | - |

> **注意**: 湿度データは直近3ヶ月のみ取得可能です（Open-Meteo APIの仕様による制限）

## 使い方

### 1. 地点を選択
日本全国の主要都市から、比較したい地点を最大3つ選択します。

### 2. 期間を設定
比較したい期間を指定します（過去20年分のデータから選択可能）。

### 3. 比較実行
「比較する」ボタンをクリックすると、各地点の気象データがグラフとテーブルで表示されます。

## 技術スタック

- **TypeScript**: 型安全な開発
- **Vite**: 高速なビルドツール
- **Chart.js**: グラフ描画
- **Open-Meteo API**: 気象データ提供元

## 開発環境のセットアップ

### 必要な環境
- Node.js
- npm

### インストール手順

```bash
# リポジトリをクローン
git clone https://github.com/gett0728/weather-comparison.git
cd weather-comparison

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスしてください。

## ビルド

```bash
# 本番用ビルド
npm run build

# ビルド結果をプレビュー
npm run preview
```

## 使用例

### 移住検討の活用シーン

- **冬の寒さを比較**: 雪が少ない地域を探したい
- **夏の暑さを確認**: 猛暑が厳しくない地域を選びたい
- **降水量を調査**: 雨が少ない/多い地域を知りたい
- **日照時間を比較**: 晴れの日が多い地域に住みたい
